Beranda Materi Quiz Kuisioner UX

1. Apa itu WWW?

WWW (World Wide Web) adalah sistem informasi global yang menggunakan internet untuk mengakses dokumen dan sumber daya melalui browser. Contohnya seperti membuka situs web (seperti Google, YouTube, Wikipedia, dll).

2. Apa itu Google?

Google adalah mesin pencari paling populer di dunia. Google membantu kita mencari informasi di internet. Google juga menyediakan layanan lain seperti Gmail, Google Drive, Google Maps, dan YouTube.

3. Fungsi Zoom

Zoom adalah aplikasi konferensi video yang memungkinkan kita melakukan pertemuan secara online. Cocok digunakan untuk sekolah daring, rapat kerja, atau diskusi kelompok dari jarak jauh.

4. Contoh Web Browser

Web browser adalah aplikasi untuk membuka halaman web. Contoh browser adalah Google Chrome, Mozilla Firefox, Safari, Microsoft Edge, dan Opera.

5. Fungsi Tombol Refresh

Refresh adalah fitur untuk memuat ulang halaman web. Jika halaman tidak tampil dengan sempurna atau terjadi gangguan koneksi, tombol refresh akan mencoba membuka kembali halaman yang sama.
